Security Notice

Please be assured that your information will be kept secure. We take the following measures:

Deep JavaScript Safety Procedures

We implement multiple layers of security, including:

  • Data Encryption: All sensitive data is encrypted during transmission using HTTPS protocols.
  • Validation: Input fields are validated on the client-side to prevent injection attacks.
  • Session Management: We use secure tokens for managing user sessions to mitigate session hijacking.

Internal and External Safety JavaScript

Our JavaScript includes:

  • Sanitization: All inputs are sanitized to prevent XSS (Cross-Site Scripting) attacks.
  • Captcha: We utilize captcha to prevent automated bots from submitting donation forms.
  • Logging: All suspicious activities are logged for monitoring and review.
